# 🛍️ STYKE – Rent Luxury Clothes Online

STYKE is a full-stack e-commerce platform for renting luxury clothing. It offers a user-friendly interface for customers and admin-level controls for managing inventory.

---

## 📌 Features

- 👤 User Registration & Login (with authentication)
- 🛒 Browse luxury dresses by category
- 📦 Add to Cart and Checkout
- 🧑‍💼 Admin panel to manage products and users
- 📱 Responsive layout (mobile-friendly)

---

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: HTML, CSS, JavaScript (EJS Templates)
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Environment**: dotenv for config
